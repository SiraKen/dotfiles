# Neovim

## Basic Usage

Default leader key is `<space>`.

```
# Toggle buffer
[ + b -> Previous buffer
] + b -> Next buffer

# Toggle fuzzy finder
<space> <space>
```

To find more keymaps, visit [Keymaps](https://www.lazyvim.org/keymaps).
